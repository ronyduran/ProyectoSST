¿Qué es S.S.T.?

Smart Sanitizing Tunnel: " Este proyecto consiste en un túnel higienizante, en cual se activará de forma automática, es decir, sin tener ninguna interacción por parte del usuario, por tanto, funcionara de manera autónoma, además, se buscará que cada persona que vaya a ingresar al local donde este está instalado, porte mascarilla, como una medida de protección y prevención. También, mediante este túnel se podrá saber el flujo de personas que pasa a través de él, así como los datos básicos de estas personas, lo cual puede servir como información a analizar para el local, si así lo desea."
